A sampa-team é a organização responsável pelo desenvolvimento colaborativo do SAMPA, um projeto acadêmico e tecnológico voltado ao monitoramento ambiental de ambientes de armazenamento de alimentos.

<ul>
  <li>Desenvolva e mantenha o firmware do ESP32 para coleta de dados ambientais;</li>
  <li>Gerencie o backend, responsável pelo processamento, armazenamento e análise das informações;</li>
  <li>Construa o frontend, fornecendo dashboards e visualizações para usuários;</li>
  <li>Organize a infraestrutura necessária para deploy, integração contínua e escalabilidade.</li>
</ul>

A sampa-team busca promover boas práticas de desenvolvimento, documentação clara e colaboração eficiente, servindo como base para projetos acadêmicos, pesquisas em IoT e soluções inovadoras de monitoramento ambiental.
